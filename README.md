# twitter_api_test
Playing with twitter API


### How do I get set up? ###
Step 1 - Clone Repo    
Step 2 - `npm install`    
Step 3 -  Set Twitter key/secret at `/util/twitter.js`    
Step 4 -  Set Mongodb connection at `/util/mongo.js`    
Step 5 -   `npm start`    

App should be running at:   
`http://localhost:3000/`
